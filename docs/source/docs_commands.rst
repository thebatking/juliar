=================

=================

\*absolute \*
----------------
Description: Takes in the numbers, and outputs the absolute numbers.

Input: Numbers

Arguments: NONE

Example(s):
``*absolute -1 2 -3 -5 50 *``

Returns:
``1 2 3 5 50``

----------------

Input: A decimal

Arguments: ``degrees`` or ``360`` or True value (Optional: whether to use degrees [default uses radians])

``1.0471975511965979``
